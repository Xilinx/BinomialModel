# Binomial Model Project: P00_sw

This project represents a SW version of the Binomial Model to be run on CPU with a single or multiple threads. The results obtained in this project will be compared against the results obtained by running the application on the Alveo card.

Please refer to the [BinomialModel.pdf] document for detailed information regarding design setup, execution and results comparison.
